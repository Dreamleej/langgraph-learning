## 🚀 快速开始

### 1. 环境准备
```bash
# 安装额外依赖
pip install fastapi uvicorn python-multipart
pip install langsmith langchain-openai
pip install chromadb sentence-transformers
pip install pillow speechrecognition
```

### 2. 选择案例
```bash
# 本地服务器
cd local_server && python main.py

# LangSmith集成
cd langsmith_integration && python monitoring_example.py

# RAG系统
cd rag_systems && python retrieval_example.py
```
