## 💡 关键概念

### State（状态）
状态是LangGraph工作流中的数据容器，定义了在工作流中流动的数据类型和结构。

### Node（节点）
节点是执行具体操作的函数，接收当前状态并返回更新后的状态。

### Edge（边）
边定义了节点之间的连接关系和执行顺序。

### StateGraph（状态图）
状态图是包含节点、边和状态定义的完整工作流。
